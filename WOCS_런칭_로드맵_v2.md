### 미드저니 작업 방법 (초보용)
```
1. PC에서 WOCS_미드저니_프롬프트_v3.txt 연다
2. 크롬으로 midjourney.com 접속 → 로그인
3. [1]번 프롬프트를 복사 (Ctrl+C)
4. 미드저니 입력창에 붙여넣기 (Ctrl+V) → Enter
5. 30초~1분 기다림 → 이미지 4장 생성
6. 마음에 드는 이미지 클릭 → Upscale → 다운로드
7. 파일명: 01-ex15-ext.png 으로 저장
8. [2]번 프롬프트로 반복
```

