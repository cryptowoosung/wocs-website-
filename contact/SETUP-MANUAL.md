# WOCS 무인 리드 수집 시스템 — 세팅 매뉴얼

## 전체 구조
```
고객 (ROI 계산기에서 폼 제출)
  ↓ fetch POST
Google Apps Script (무료 서버)
  ↓ 동시 실행
  ├─ 구글 시트에 DB 자동 기록
  ├─ 고객 폰으로 "접수 완료" 문자
  └─ 내 폰으로 "신규 DB" 알림 문자
```
서버 비용: **0원** (GAS 무료, 하루 2만 건 처리 가능)

---

## STEP 1: 구글 스프레드시트 생성 (1분)

1. https://sheets.google.com → **빈 스프레드시트** 만들기
2. 시트 이름(하단 탭)을 `leads`로 변경
3. URL에서 **시트 ID** 복사:
   ```
   https://docs.google.com/spreadsheets/d/여기가_시트_ID/edit
   ```
   → `여기가_시트_ID` 부분만 복사해 메모장에 붙여넣기

---

## STEP 2: GAS 코드 입력 (3분)

1. 시트 상단 메뉴 → **확장 프로그램** → **Apps Script** 클릭
2. 기본 코드(`function myFunction(){}`) 전부 삭제
3. `gas-backend.gs.txt` 파일 내용 **전체 복사-붙여넣기**
4. 상단의 **CONFIG** 부분 수정:
   ```javascript
   SHEET_ID: '1단계에서 복사한 시트 ID 붙여넣기',
   ```
5. **Ctrl+S** 로 저장 (프로젝트 이름: `WOCS-Lead`)

---

## STEP 3: GAS 웹앱 배포 (2분)

1. Apps Script 에디터 우측 상단 → **배포** → **새 배포**
2. 톱니바퀴 아이콘 → **웹 앱** 선택
3. 설정:
   - 설명: `WOCS 리드 수집`
   - 실행 사용자: **본인**
   - 액세스 권한: **모든 사용자** (⚠ 중요! "Anyone"으로 해야 외부 접근 가능)
4. **배포** 클릭
5. 권한 요청 팝업 → **고급** → **WOCS-Lead(안전하지 않음)로 이동** → **허용**
6. **웹 앱 URL** 복사 (형태: `https://script.google.com/macros/s/AKfyc.../exec`)

---

## STEP 4: 프론트엔드에 URL 연결 (30초)

1. `roi-calculator.html` 파일 열기
2. 상단의 이 줄을 찾기:
   ```javascript
   var GAS_URL = 'YOUR_GAS_WEB_APP_URL_HERE';
   ```
3. `YOUR_GAS_WEB_APP_URL_HERE` → 3단계에서 복사한 **웹앱 URL**로 교체
4. 저장

---

## STEP 5: 테스트 (1분)

### 방법 A: GAS 에디터에서 직접 테스트
1. Apps Script 에디터로 돌아가기
2. 상단 함수 선택: `testDoPost` 선택
3. ▶ (실행) 클릭
4. 구글 시트에 테스트 행이 추가되었는지 확인

### 방법 B: 실제 홈페이지에서 테스트
1. `roi-calculator.html`을 브라우저에서 열기
2. 시뮬레이션 값 입력
3. 하단 리드 폼에 테스트 이름/전화 입력 → 전송
4. 구글 시트에 데이터가 기록되었는지 확인

---

## STEP 6: 알리고 문자 연동 (선택, 10분)

> 문자 없이도 시트 DB 수집만으로 충분히 작동합니다.
> 문자 연동은 나중에 해도 됩니다.

1. https://smartsms.aligo.in 회원가입
2. **API Key** 발급: 마이페이지 → API 연동 → API Key 복사
3. **발신번호 등록**: 010-4337-0582 등록 (본인 인증 필요)
4. GAS 코드의 CONFIG 수정:
   ```javascript
   ALIGO_API_KEY: '발급받은 API Key',
   ALIGO_USER_ID: '알리고 로그인 ID',
   ALIGO_SENDER: '010-4337-0582',
   SMS_ENABLED: true    // ← false를 true로 변경
   ```
5. Apps Script에서 다시 **배포** → **배포 관리** → **새 버전** 배포

### 알리고 요금
- SMS (90자 이내): 건당 9.9원
- LMS (2,000자 이내): 건당 30원
- 충전식: 1만원 충전하면 약 1,000건 발송 가능

---

## 트러블슈팅

| 문제 | 원인 | 해결 |
|------|------|------|
| 시트에 데이터가 안 쌓임 | 웹앱 URL 틀림 or 배포 안 함 | STEP 3 재확인, URL 정확히 복사 |
| CORS 에러 | 액세스 권한이 '본인만' | **모든 사용자**로 변경 후 재배포 |
| 권한 팝업 안 뜸 | 구글 계정 문제 | 시크릿 모드에서 다시 시도 |
| 문자 안 감 | SMS_ENABLED가 false | `true`로 변경 후 **재배포** |
| 알리고 "발신번호 미등록" | 발신번호 인증 미완료 | 알리고에서 발신번호 등록 완료 |

## 완성 후 동작 흐름

```
1. 고객이 ROI 계산기에서 시뮬레이션 실행
2. "상세 견적 받기" 폼에 이름·전화번호 입력
3. [전송] 버튼 클릭
4. → GAS 웹앱이 데이터 수신
5. → 구글 시트 'leads' 탭에 자동 기록
6. → 고객에게 "접수 완료" 문자 발송
7. → 내 폰에 "🔔 신규 DB" 알림 문자 수신
8. → 프론트엔드: "✅ 접수 완료!" 화면 표시
```

**24시간 무인 영업 시스템 완성. 서버비 0원.**
